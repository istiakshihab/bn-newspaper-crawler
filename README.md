# bn-newspaper-crawler

Bangla Newspaper Crawlers written using Scrapy.

Newspaper Covered So far:
- Amader Shomoy
- Bangladesh Today
- Bangla Tribune
- BDNews24
- Daily Nayadiganta
- Bangla Dailystar
- Ittefaq
- Janakantha
- Kalerkantho
- TBS Bangla

Dependencies:
- Scrapy
- Newspaper3k (Modfified)
